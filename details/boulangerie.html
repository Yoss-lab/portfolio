bakery website

The project involved developing a website for a bakery using HTML, Sass and JavaScript technologies. The initial mock-up was provided as a PSD file, and required careful cutting work to accurately reproduce the original design. Throughout the development, challenges were encountered, including image optimization and the implementation of gallery functionality. The end result is a functional and visually appealing website that perfectly matches the original layout.
